# 🍸 Cocktails GSAP Project

A modern, interactive frontend project built with **React**, **GSAP**, **Tailwind CSS**, and **Vite** — showcasing animated cocktail-themed UI and smooth transitions.

---

## 🚀 Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- ✨ GSAP (GreenSock)
